## 1. 历史波动统计研究

**目标**：用过去3年的XAU数据,验证当前 v5 参数(强平覆盖$86)的实战可行性。

**待研究**：
- [ ] 统计过去3年 XAU **单日**最大价格波动幅度,筛出超过 **$86** 的天数
- [ ] 对这些"超限日"逐根M1 K线分析:
  - [ ] 标注具体时间(年-月-日 时:分)
  - [ ] 标注连续单向波动的K线序列(例: 价格从X→Y, 持续N根K线)
  - [ ] 单根K线最大波动($)
  - [ ] 连续N根K线累计波动($)
- [ ] 输出"L11被打满"的潜在场景统计:
  - [ ] 多长时间内累计$86? (1min? 5min? 30min?)
  - [ ] 这种行情通常出现在什么时段(亚盘/欧盘/美盘/数据时段)?
  - [ ] 有无可识别的事件标记(NFP/CPI/FOMC/突发新闻)?

**意义**:
- 评估50%总亏强平触发的实际频率
- 决定是否需要扩展层数 / 调整距离序列 / 加入新闻时段过滤
- 倒推合适的账户本金水平(覆盖N倍历史最坏情况)

**数据源候选**:
- MT5 历史数据导出 (品种属性 → 显示历史数据)
- TradingView / TwelveData / Alpha Vantage API
- Dukascopy 免费tick数据

**输出形式**:
- CSV 或 JSON: `{date, max_minute_range, max_cumulative_range, duration, session}`
- 简要报告: 频率统计 + 典型场景描述
